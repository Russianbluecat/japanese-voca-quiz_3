# 🇯🇵 日本語 単語クイズ

일본어 단어 학습을 위한 인터랙티브 퀴즈 웹앱입니다.  
구글 시트를 데이터 소스로 사용하며, GitHub Pages로 배포됩니다.

🔗 **[라이브 데모 보기](https://russianbluecat.github.io/japanese-voca-quiz_3/)**

---

## 📌 기능

### 두 가지 퀴즈 모드

| 모드 | 설명 |
|------|------|
| 🔗 **의미 연결하기** | 4개의 단어와 의미를 클릭으로 매칭 |
| 📝 **빈칸 채우기** | 예문 속 빈칸에 알맞은 단어를 4지선다로 선택 |

### 공통 기능
- ⏱ **라운드당 20초 제한** — 타이머 게이지 + 숫자 표시, 5초 이하 시 위험 색상으로 전환
- 🔀 **랜덤 출제** — 전체 단어 풀에서 중복 없이 랜덤 추출
- 📊 **실시간 정답/오답 카운트**
- 🏁 **10라운드(문제) 후 최종 결과 화면**

### 의미 연결하기
- 라운드당 4개 단어 출제
- 시간 초과 시 미연결 쌍 자동 정답 표시 + 오답 처리
- 오답 연결 시 흔들림 애니메이션 + 빨간 선 표시

### 빈칸 채우기
- 문장 속 빈칸에 알맞은 단어 선택
- 정답 후 예문 / 히라가나 / 번역(파파고) 제공
- 시간 초과 시 2초 후 자동으로 다음 문제 이동

## 🗂 데이터 구조

구글 시트 `Raw_Data` 시트 기준:

| 열 | 컬럼명 | 설명 | 예시 |
|----|--------|------|------|
| A | `Original` | 일본어 단어 (한자/가나) | 呼ばれる |
| B | `Hiragana` | 히라가나 표기 | よばれる |
| C | `Meaning` | 한국어 의미 | 불리다, ~라고 불리다 |
| D | `Sample_Sentence` | 예문 (일본어) | 私は友達に「先生」と呼ばれます。|
| E | `Sample_Sentence_hiragana` | 예문 히라가나 | わたしはともだちに「せんせい」と… |
| F | `Blank_Sentence` | 빈칸 예문 | 私は友達に「先生」と(呼ばれ)ます。|
| G | `Translation` | 예문 한국어 번역 | 나는 친구들에게 '선생님'이라고 불린다. |

> **빈칸 채우기** 모드는 `Blank_Sentence` 열의 괄호 `( )` 또는 `（ ）` 안의 단어를 정답으로 사용합니다.

## 🚀 배포 방법

### 1. 구글 시트 공개 설정
구글 시트 → **파일 → 공유 → 링크가 있는 모든 사용자 → 뷰어**

### 2. GitHub Pages 배포
```
1. 이 레포지토리를 Fork 또는 Clone
2. index.html 파일을 수정 (SHEET_ID를 본인 시트 ID로 변경)
3. Settings → Pages → Branch: main / (root) → Save
4. https://[아이디].github.io/[레포명]/ 으로 접속
```

### 3. 시트 ID 변경
`index.html` 상단 JS 설정값:
```javascript
const SHEET_ID   = '여기에_본인_구글시트_ID';
const SHEET_NAME = 'Raw_Data'; // 시트 이름
```
